## 说明

这是`python2.7`版本的`SimpleHTTPServerWithUpload`.

1. [bones7456](http://luy.li/2010/05/15/simplehttpserverwithupload/) 同学在`python2.7`的[SimpleHTTPServer](https://docs.python.org/2/library/simplehttpserver.html#module-SimpleHTTPServer)基础上添加了上传功能.

2. [BUPTGuo](http://buptguo.com/about-me/) 同学完成了[一些改进](http://buptguo.com/2015/11/07/simplehttpserver-with-upload-file/).

3. 在`python2.7`中可以支持局域网中的文件的上传和下载.
